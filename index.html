🔐 SHA‑256 & HMAC

Tool to compute SHA‑256 and HMAC‑SHA256 hashes for multiple inputs at once. Each line is treated as a separate data item. Supports various input encodings: UTF‑8, UTF‑16, Hex, and Base64.

💡 Each line is processed as one data item. Empty lines, spaces, and tabs are included. SHA‑256 hash updates automatically as you type.
⚙️ Choose the encoding that matches your data format.
